# Java Study

Repositório pessoal para reunir projetos, exercícios e experimentos feitos durante meus estudos de Java.

A ideia aqui é manter tudo organizado em um só lugar, separando cada projeto por pasta para facilitar a consulta, a evolução e a comparação entre diferentes abordagens.

## Estrutura atual

```text
java-study/
├── JavaFK/
├── http-java/
└── springboot/
```

- `JavaFK/`: projeto JavaFX com Maven.
- `http-java/`: pasta reservada para estudos relacionados a HTTP e aplicações Java.
- `springboot/`: pasta reservada para estudos com Spring Boot.

## Objetivo do repositório

Este repositório serve como base para:

- guardar projetos de estudo;
- testar ideias e conceitos;
- praticar Java e tecnologias do ecossistema;
- registrar a evolução ao longo do tempo.

## Tecnologias que podem aparecer aqui

Dependendo do projeto, este repositório pode conter estudos com:

- Java;
- Maven;
- JavaFX;
- Spring Boot;
- APIs HTTP;
- banco de dados;
- FXML e interfaces gráficas.

## Como executar o projeto JavaFK

O projeto `JavaFK` usa Maven e Java 21.

No terminal, entre na pasta do projeto e execute:

```bash
./mvnw javafx:run
```

No Windows, se preferir:

```bat
mvnw.cmd javafx:run
```

## Como organizar novos projetos

Quando eu adicionar um novo estudo, a ideia é seguir este padrão:

- criar uma pasta própria para cada projeto;
- manter um `README.md` interno quando o projeto exigir instruções específicas;
- deixar o nome da pasta o mais claro possível;
- separar por tema quando fizer sentido, como `springboot`, `http-java`, `junit`, `jdbc` e assim por diante.
